# Ueats Automation Test Project

This project contains automated tests for Ueats, a Command Line Interface (CLI) tool developed as part of our pytest udemy course.

## Prerequisites

Before running the tests, ensure you have the following installed:

- Python > 3.8
- ueats cli
- pytest
- pytest-mock

## Running the Tests

To run the automated tests, navigate to the project root directory and run
   ```bash
   pytest
